
# Sigma chat

Chat with everyone online, multiple channels, message saving, deleting and secure accounts.


## Channels


#### Get channel

```http
  GET /msg/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `int` | **Required**. Id of item to fetch |
